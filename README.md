# pRojEctCaSE 🧪

A clean, developer-grade text transformation and encoding laboratory with a high-fidelity doodle/sketch aesthetic. Built for engineers who love efficient tools that feel creative and organic.

![Aesthetic](https://img.shields.io/badge/Aesthetic-Hand--Drawn-marker-red)
![Stack](https://img.shields.io/badge/Stack-Next.js%20|%20TS%20|%20Tailwind-blue)

## ✨ Features

- **Encoders & Decoders**: Base64, URL, ROT13, Caesar, Binary, Hex, and Morse.
- **Smart Decode**: One-click auto-detection of common encoding formats.
- **Text Transformers**: Case changes, line numbering, regex find/replace, and whitespace cleanup.
- **ASCII Art**: Figlet-powered text-to-art generation with multiple fonts.
- **Dev Tools**: JSON, HTML, and CSV escaping utilities.
- **Generators**: Strong passwords, UUIDs, Short IDs, and academic-style Roll Numbers.
- **Pure Logic**: All core functions are written as pure TypeScript functions, separated from the UI.

## 🎨 Design System

pRojEctCaSE celebrates "authentic imperfection":
- **No Straight Lines**: Every container uses irregular wobbly borders.
- **Paper Texture**: Background simulates a physical engineering notebook.
- **Haptic Interaction**: Buttons and cards use solid offset "hard shadows" that animate to simulate physical pressing.
- **Typography**: Uses `Kalam` and `Patrick Hand` for the sketched look, paired with `JetBrains Mono` for developer output.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📂 Project Structure

- `src/core`: Pure TypeScript logic for all transformations.
- `src/components`: Reusable hand-drawn UI components (SketchButton, SketchCard, etc.).
- `src/app`: Next.js App Router pages and global styles.

## 🛠 Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Art**: Figlet / Rough.js
